# Instagram Report Bot


*****************************************************************
** Do not modify the reportBotWithGUI.py file **
*****************************************************************


-----------------------------------------
------------- Instructions --------------
-----------------------------------------

1. Make sure you have Python installed and that you added Python to PATH during the installation
   (version 3.8.5 was used when creating the bot).
2. Run the install_packages.bat file 
   (This will install the needed packages to run the bot.)
3. Install and update Google Chrome.
4. Run the run_script.bat file to run the Report Bot. 
5. Make sure to follow the instructions that are presented when the Bot opens.
